ShadeUI is fully themeable and powered by design tokens built on top of Tailwind CSS. Our theming system ensures consistency, flexibility, and easy customization across all components.



## ðŸŒˆ How Theming Works

- **Design Tokens:** Centralized variables for color, typography, spacing, and shadows.

- **CSS Variables:** Tokens are exposed via CSS custom properties.

- **Tailwind CSS Integration:** Seamless compatibility with your existing Tailwind setup.

- **Dark Mode Support:** Built-in light and dark mode switching.
